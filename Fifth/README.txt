В задании запрещено использовать stream

1. Работа с коллекциями:
	1.1. Найти в тексте все использованные слова и поместить их в коллекцию Set.
		Необходимо прочитать текстовый файл и поместить текст из данного текстового файла в строку. Далее данную строку необходимо 
		обработать таким образом, чтобы из неё можно было бы считывать слова. Далее каждое полученное слово поместить в Set.
		В конце выводим в консоль количество использованных слов.
	1.2. Найти в тексте топ N (настраиваемое значение) слов и вывести количество этих слов.
	Отсортировать по количеству. Распечатать в консоль. 
	Из получившегося отсортированного набора данных вывести первые N записей.
2. Написать интерфейс ISearchEngine. Это будет интерфейс в котором будут методы:
	2.1. long search(String text, String word) - Метод возвращает информации о количестве раз которое встречается слово в тексте.
	Первый параметр текст в котором ищем, второй параметр что ищем в тексте.
3. Реализовать интерфейс ISearchEngine:
	3.1. Написать класс EasySearch. Реализовать поиск используя метод indexOf из класса String.
	В данной реализации запрещено использовать регулярные выражения в любом виде, для любых задач.
	Данный класс ищет слова с учётом регистра.
	3.2. Написать класс RegExSearch реализующий интерфейс ISearchEngine. Реализовать поиск при помощи класса Matcher.
	Данный класс ищет слова без учёта регистра.
	3.3. Написать декоратор SearchEnginePunctuationNormalizer для ISearchEngine который будет удалять нежелательные символы.
	Любые знаки препинания, лишние пробелы и например переводы строк.
