SiteLoader. Класс подключения к :
1. при подключение к сайту может быть ошибка, это может быть проблема с сайтом, скоростью его ответа или с недоступностью интернета. 
2. В классе NBRBLoader требуется дописать метод handle. При помощи данного метода в строке content вы дожны найти курс валюты,
имя которой передали в параметре currencyName. Почти обычный поиск строки currencyName по строке content 
(нельзя использовать регулярные выражения).
3. Написать класс наследника SiteLoader по примеру NBRBLoader для Альфа-банк (https://developerhub.alfabank.by:8273/partner/1.0.0/public/rates)
4. Реализовать сохранение полученного или полученных курсов в файл. При запуске программы дать возможно пользователю указывать путь к файлу
для сохранения курса, иначе курс должен сохраняться рядом рядом с запускаемой программой.
5. Если по выбранному нами пути уже существуют сохранённые курсы необходимо дописать данные в существующий файл.
